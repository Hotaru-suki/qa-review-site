# Linux

## 摘要

Linux 是测试、测开和后端排障中的基本环境能力。这个专题不打算把 Linux 写成系统教材，而是围绕最常见、最能直接提升排查效率的能力来组织，包括文件操作、进程服务、网络端口和日志排障。

---

## 专题结构

- `常用命令与文件操作`
  处理文件查找、内容查看和文本筛选。
- `进程与服务管理`
  处理进程状态、服务启动和资源观察。
- `网络与端口排查`
  处理连通性、监听状态和连接定位。
- `日志与排障路径`
  处理日志阅读和基础排障流程。

---

## 阅读重点

- 先建立“先看什么、后看什么”的排障顺序
- 再把命令能力和进程、网络、日志这几个方向串起来
