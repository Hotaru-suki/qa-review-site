# SQL 基础与查询

## 摘要

这一组内容解决“数据到底怎么查”，重点是把查询目标稳定地翻译成 SQL。真正要掌握的不是零散语法，而是结果集怎样被过滤、分组、连接和重组。

---

## 本组内容

- `SQL 基础语法`
  处理单表查询、过滤、排序和聚合。
- `连接与子查询`
  处理多表组合和嵌套查询。
- `窗口函数与高频题`
  处理排名、分组编号、每组最新一条等场景。
- `测试场景 SQL`
  处理测试和排查里的实用查询。

---

## 阅读重点

- 一条 SQL 是怎样一步步把结果集加工出来的
- 多表关系、排名、分组编号这类问题怎么稳定处理
- SQL 在测试和排查里怎么落地使用
