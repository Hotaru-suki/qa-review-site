## 学习目标

通过本专题，你应逐步具备以下能力：

- 能编写常见单表、多表查询 SQL
- 能理解连接、分组、子查询、窗口函数等写法
- 能使用 SQL 进行数据验证、结果排查和问题定位
- 能掌握事务、锁、隔离级别、索引等核心知识
- 能应对测试 / 测开 / 后端相关岗位中的数据库面试问题

---

## 学习建议

建议按照以下顺序学习：

1. 先学 SQL 基础语法与数据库基础概念
2. 再学连接、子查询、分组、窗口函数
3. 结合测试场景练习 SQL 查询与数据校验
4. 最后补事务、索引、慢查询与高频面试题

---

## 适用场景

- 软件测试 / 测试开发岗面试复习
- 接口测试与数据库校验
- 后台数据排查与问题定位
- 后端基础知识补强
- SQL 专项训练

---

## 学习提示

学习数据库不要只停留在“记语法、背概念”，更重要的是形成三种能力：

- **会写 SQL**
- **会查数据**
- **会讲原理**

既要能写出查询语句，也要能结合业务场景解释查询思路、定位问题原因，并在面试中表达清楚。
