# C++

## 摘要

C++ 专题按“基础语法 -> 对象与内存 -> STL”三条主线组织，重点不是堆语法点，而是建立对象语义、资源管理和容器算法配合这三层共同理解。

---

## 本专题结构

### 基础语法

- 引用、指针与基础语法

### 对象模型与内存

- 对象生命周期管理
- 内存泄漏与排查
- 拷贝、移动与资源所有权

### STL 与高频语法

- 模板、string、pair、lambda、优先队列
- STL 容器、迭代器与算法

---

## 本专题重点

- 基础语法先解决“能看懂、能解释”
- 对象与内存解决“为什么会这样执行”
- STL 解决“容器怎么选、算法怎么配”
