# C++ 对象模型与内存

## 摘要

这部分是 C++ 最容易拉开差距的地方。理解对象生命周期、资源管理和内存问题，是从“会写代码”走向“会解释行为和排查问题”的关键。重点不是多背术语，而是把对象创建、拷贝、移动、销毁和资源释放放回同一条生命周期里理解。

---

## 本组内容

- `对象生命周期管理`
  解决对象从创建到销毁的行为过程。
- `内存泄漏与排查`
  解决资源没释放、释放时机不对和排查路径问题。
- `拷贝、移动与资源所有权`
  解决值语义、移动语义和所有权转移问题。

---

## 阅读重点

- 先搞清对象什么时候构造、拷贝、析构
- 再理解资源所有权为什么决定安全性和性能
