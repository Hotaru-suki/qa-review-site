# 文件系统与 IO

## 摘要

操作系统除了管理进程和内存，还要管理文件如何组织、数据如何读写、进程如何等待 IO 结果。这一组关注文件系统和 IO 模型如何影响程序行为与并发能力。

---

## 本组内容

- `文件系统核心概念`
  解决文件、目录、缓存和持久化的基本认知。
- `阻塞 IO 与多路复用`
  解决 IO 等待、事件通知和高并发网络处理方式。
- `磁盘 IO、页缓存与回写`
  解决数据为什么不是立刻落盘以及写入抖动从哪里来。
- `inode、文件描述符与打开文件表`
  解决进程打开文件后，内核里到底维护了哪些对象。
- `mmap、普通读写与缺页异常`
  解决“映射文件更快吗”这类常见但容易答浅的问题。

---

## 阅读重点

- 文件系统看数据组织和读写路径
- IO 看等待模型和并发方式
