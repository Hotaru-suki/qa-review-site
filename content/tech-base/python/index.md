# Python

## 摘要

Python 专题关注测试开发和脚本场景里最常用的语言基础，内容会沿着“数据结构 -> 流程组织 -> 文件与标准库 -> 异常与对象”逐步展开。

---

## 本专题结构

### 数据类型

- 数据类型总览与区别
- 数据类型常用操作

### 流程控制与函数

- 条件、循环与推导式
- 函数、作用域与参数传递

### 文件处理与常用标准库

- 文件、JSON 与脚本输入输出
- 高频标准库怎么理解和选用

### 异常处理与类

- 异常处理在解决什么问题
- 类、对象与面向对象怎么理解

---

## 本专题重点

- 先理解数据结构和可变性
- 再理解逻辑如何被函数、异常和类组织起来
- 最后把语言基础落到真实脚本和自动化场景里
