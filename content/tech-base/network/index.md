# 计算机网络

## 摘要

网络专题关注请求为什么能到达、为什么会失败、为什么会慢。内容会按“传输层语义 -> HTTP / HTTPS / DNS -> 网络故障与定位”三条主线展开。

---

## 本专题结构

### 传输层与连接语义

- TCP 与 UDP 的核心区别
- TCP 三次握手、四次挥手与可靠性

### HTTP、HTTPS 与 DNS

- 从输入 URL 到页面显示
- Cookie、Session 与 Token 怎么区分

### 网络故障与性能定位

- 延迟、超时与重试怎么串起来看
- 网络问题如何按层定位

---

## 本专题重点

- 先理解连接和传输到底在解决什么
- 再把 HTTP、HTTPS、DNS 放回整条请求链路
- 最后学会按层分析网络故障和时延问题
