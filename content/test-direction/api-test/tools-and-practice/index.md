# 工具与实践

## 摘要

这一组内容关注接口测试怎么真正落地，包括调试、自动化、数据校验、Mock 和性能工具。重点不是认识工具名，而是知道这些工具分别解决哪一段测试工作。

---

## 本组内容

- `Postman / Apifox 调试`
  解决联调和快速验证阶段的问题。
- `pytest + requests 接口自动化`
  解决批量回归和持续执行问题。
- `JMeter 接口性能测试`
  解决性能验证和升压分析问题。
- `MySQL / Redis 数据校验`
  解决结果闭环问题。
- `Mock 在接口测试中的应用`
  解决依赖未就绪和场景构造问题。

---

## 阅读重点

- 调试阶段如何快速构造请求和验证响应
- 自动化阶段如何组织用例、断言和数据驱动
- 联调受阻时如何用 Mock 提前推进测试
- 接口校验为什么必须结合数据库、缓存和日志链路
